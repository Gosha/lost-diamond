The Lost Diamond
================
Student project. In implementation of the board game "The Lost Diamond".
There's a map editor and a game.

Usage
=====
Run the game

    ant
    java -jar Game.jar
    
Run the editor

    cd bin
    java se.liu.ida.geoza435.tddc69.project.GUI.editor.Editor
